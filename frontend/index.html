          Pruebas

          Crear prueba

          Crea una prueba extraordinaria. La sala y el bloque se reservan despues en la pantalla correspondiente.

          Flujo actual: primero crea la prueba y luego usa "Reservar sala y bloque" para asignarle fecha, bloque y sala.

          Pruebas

          Reservar sala y bloque

          Reserva una sala y un bloque para una prueba existente, y asigna automaticamente a todos los alumnos inscritos.

          Si algun alumno tiene tope o la sala no tiene cupos suficientes, la prueba no se agenda y no se crean asignaciones.
