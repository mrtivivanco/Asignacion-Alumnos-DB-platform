            Pruebas

            Crear prueba

            Crea una prueba extraordinaria y reserva la sala/bloque que usara. Esto aun no asigna alumnos.

            Flujo actual: este formulario crea la prueba y reserva inmediatamente su sala y bloque.

            Pruebas

            Reservar sala y bloque

            Reserva una sala para una prueba existente en su bloque correspondiente. Esta accion no asigna alumnos.

            La API valida que el bloque elegido corresponda al bloque definido para la prueba seleccionada.

              PDFs de pruebas

              Carga, revisa, descarga o elimina el PDF asociado a cada prueba. Los archivos se guardan en MongoDB/GridFS.
